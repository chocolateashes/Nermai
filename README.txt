Nermai Web Application

-Must have Python3
-Pip must be installed
	To install on Windows, download and open get-pip.py
	To install on Mac OS X & Linux, run
		$ curl -O https://bootstrap.pypa.io/get-pip.py
		$ python get-pip.py --user
-Install Flask
	$ pip install flask

————————

To Run Nermai
-Download zip file Nermai.zip
-Navigate to Nermai directory in terminal or command prompt
-Run python hello.py (if that doesn’t work try python3 hello.py)
-At this point the terminal should say something like
	*Running on http://127.0.0.1:5000/
-Open that address in your browser (localhost:5000)
